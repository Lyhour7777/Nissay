## Installation
Note: Require Python version $>=$ 3.11.x

- navigate to terminal or cmd at your desire directory then:<br>
> `git clone https://github.com/Lyhour7777/Nissay.git` <br>
- Open folder in your coding workplace (e.g `D:/NissayProject/`) <br>
- Create virtual environment and Install librabry<br>
> `python -m venv .venv` <br>
> `.venv\Scripts\Activate.ps1`<br>
> `pip install -r requirements.txt`<br>
- After finish installation, run steamlit frontend: <br>
> `streamlit run app.py`
