# Nissay
This project aim to identify human feeling.

## Features
- Detect feeling by audio
- Detect feeling by audio

## Installation/Dev
Note: Require Python version $>=$ 3.11.x

1. Clone Repo
```bash
git clone https://github.com/Lyhour7777/Nissay.git
cd NissayProject
```

2. Optional: Create virtual environment
```bash
python -m venv .venv
.venv\Scripts\Activate.ps1
```

3. Install dependencies
```bash
pip install -r requirements.txt
```

4. Streamlit:
```bash
streamlit run app.py
```
